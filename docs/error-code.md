## 운영 원칙

1. 코드 추가/수정은 시트에서 먼저 반영합니다.
2. 코드 반영 시 `src/main/java/com/gachi/be/global/code/ErrorCode.java`와 동기화합니다.
3. PR에는 변경된 코드와 설명(의도/상태코드/메시지)을 함께 남깁니다.
